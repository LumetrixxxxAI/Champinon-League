# 🍄 Champiñón League

Web oficial de la Champiñón League: la liga de fútbol amistoso entre amigos.

Cada jornada se graba el partido, se sacan los mejores goles, regates y patadas, y se elige un MVP. Cada 15 jornadas se cierra una temporada, y quien más MVPs acumule se lleva el **Balón de Oro**.

## Contenido

- `index.html` — Inicio y resumen de jornadas
- `clasificacion.html` — Tabla de MVPs de la temporada en curso
- `balon-de-oro.html` — Palmarés histórico del Balón de Oro
- `css/style.css` — Estilos del sitio

## Cómo actualizar los datos

Los datos de jornadas, clasificación y MVPs son de ejemplo. Para actualizarlos, edita directamente el HTML de cada página (tarjetas de jornada en `index.html`, filas de la tabla en `clasificacion.html`, tarjetas de temporada en `balon-de-oro.html`).

## Publicar con GitHub Pages

Activa GitHub Pages en **Settings → Pages**, rama `main`, carpeta `/ (root)`, y la web quedará disponible en la URL pública del repositorio.
